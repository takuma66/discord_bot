# discord_bot
VALORANTのカスタムマッチ用ディスコードbot
## コマンド一覧
* /map マップをランダムに選択
* /coin 裏か表を出力
* /team VCに入っている人をランダムに５人選択(未実装)

## ファイルの説明
### discordbot.py
* 実際に動くソースコード

### Procfile
* heroku(サーバー)で動かすコードを指定するためのファイル
* 基本的に変更する必要はない
### runtime.txt
* pythonのバージョンを指定しているファイル

### requirements.txt
* 使用している外部ライブラリをここに書く必要がある
